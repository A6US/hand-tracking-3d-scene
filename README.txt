# Hand Tracking 3D Scene

Este proyecto es una aplicación de escritorio que permite interactuar con una escena 3D de partículas usando gestos de las manos.

## Cómo funciona
El programa utiliza la cámara para detectar la distancia entre el pulgar y el índice:
- **Mano Izquierda:** Controla la rotación de la cámara.
- **Mano Derecha:** Controla el tamaño (gravedad) de la esfera de puntos.

## Requisitos
- OpenCV
- Mediapipe
- Numpy
- Pillow (PIL)
